>  Q.1번이 2초 걸리고 2번부터는 1초씩 추가로 걸린다. 어떤 문자열을 입력받았을 때 몇초가 걸리나?

```python
string = input()
str_len = len(string)
sum = 0

for i in range(str_len):
    if string[i] == 'A' or 'B' or 'C':
        sum += 3
```

> cause of failure : 처음에는 위와 같은 방식으로 실행을 했다.
